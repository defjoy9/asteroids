# Asteroids Clone

A simple Asteroids game clone built with Pygame and object-oriented programming.

## Features
- Player movement with WASD keys
- Ship rotation controls
- Shoot asteroids with SPACE bar
- Asteroids split into smaller pieces when shot
- Collision detection (game over on asteroid hit)

## How to Run
```bash
python main.py
```

## Controls
- **WASD**: Move ship
- **Left/Right Arrow**: Rotate ship  
- **SPACE**: Shoot
- **Close window**: Exit game
